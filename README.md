# 花草养护记录网站

这是一个简单易用的花草养护记录网站，可以帮助您轻松记录植物的浇水情况、查看养护历史和统计数据。

## 功能特点

- **植物选择**：提供常见植物类型，每种植物都有对应的图片
- **浇水记录**：点击按钮即可记录浇水，自动保存日期时间
- **状态描述**：可以添加植物状态描述
- **照片上传**：支持上传植物照片
- **日历查看**：显示浇水日期，点击查看详情
- **数据统计**：显示每盆植物的浇水次数
- **温湿度记录**：自动记录当前环境温湿度
- **历史记录**：查看植物的养护历史

## 如何使用

### 本地使用

1. 下载所有文件到本地文件夹
2. 直接在浏览器中打开 `index.html` 文件
3. 或者启动本地服务器：
   - 打开命令提示符/终端
   - 进入文件所在目录
   - 运行 `python -m http.server 8000`
   - 在浏览器中访问 `http://localhost:8000`

### 分享给他人

1. 将所有文件压缩为ZIP文件
2. 发送给他人
3. 他人解压后按照上述本地使用方法打开

## 技术说明

- 纯前端实现，使用HTML、CSS和JavaScript
- 数据保存在本地存储中，无需登录
- 响应式设计，适配不同屏幕尺寸
- 使用莫兰迪色系，风格清新自然

## 注意事项

- 所有数据保存在浏览器本地存储中，清除浏览器数据会导致数据丢失
- 照片上传功能会将照片以base64格式存储在本地，建议上传较小的照片
- 温湿度数据为模拟数据，实际应用中可以通过传感器或API获取
